# Habit Tracker Dashboard

A modern, responsive dashboard to track your daily habits. Built with vanilla HTML, CSS, and JavaScript, it stores all data locally in the browser and requires no backend—making it fully compatible and ready to deploy on GitHub Pages without any build steps.

## Features

- **Monthly Tracking Grid**: Track unlimited habits across every day of the month
- **Interactive Data Visualization**: A dynamic line chart via Chart.js provides visual feedback on daily completion rates
- **Dynamic Stats**: Quick overview of total habits completed and your monthly completion rate
- **Local Storage System**: Auto-saves your progress reliably in your browser
- **Dark & Light Mode**: Seamlessly switch themes based on your preference

## Tech Stack
- HTML5
- CSS3 (Vanilla + CSS Variables)
- Vanilla JavaScript
- Chart.js (via CDN)

## Usage

1. Open `index.html` in your browser.
2. Add a new habit using the top input field.
3. Tap on the grid squares to mark a habit as complete for specific days.
4. Click on a habit name to edit or delete it.
5. Watch your progress in real-time on the score graph!
